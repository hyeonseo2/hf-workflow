# Quality Skill

This folder contains a reusable Codex skill for reviewing a specific Korean
Hugging Face Blog translation file for translation quality.

## Input

The primary input is a manifest generated by `translation-flow`.

The skill should read:

- `source.url`
- `source.title`
- `translation.file_path`
- `handoff.quality.checks`

## Output

The skill may produce:

- direct fixes for clear translation issues
- a quality review note
- a scorecard covering fidelity, fluency, terminology, formatting, and links

SEO optimization belongs to `seo-skills`.

## Tool

The deterministic baseline tool lives under `tools/`.

```bash
python skills/quality/tools/simple_quality_report.py \
  --manifest reports/pr-130/manifest.yaml \
  --target-root target-repo \
  --output reports/pr-130/quality-report.md
```
