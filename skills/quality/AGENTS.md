# Quality Skills Agent Guide

Use this repo for translation quality review skills only.

Expected workflow:

1. Read the manifest generated by `translation-flow`.
2. Locate `translation.file_path`.
3. Compare the translation against the source post.
4. Review fidelity, fluency, terminology, formatting, code blocks, and links.
5. Apply focused fixes or write a review artifact.

Do not create new translation files here. That belongs to `translation-flow`.
