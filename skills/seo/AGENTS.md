# SEO Skills Agent Guide

Use this repo for SEO optimization skills only.

Expected workflow:

1. Read the manifest generated by `translation-flow`.
2. Locate `translation.file_path`.
3. Review frontmatter, title, description, headings, slug, and links.
4. Apply focused SEO edits or write a review artifact.

Do not create new translation files here. That belongs to `translation-flow`.
