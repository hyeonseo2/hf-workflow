# SEO Skill

This folder contains a reusable Codex skill for reviewing a specific Korean
Hugging Face Blog translation file for search and discoverability.

## Input

The primary input is a manifest generated by `translation-flow`.

The skill should read:

- `source.url`
- `source.title`
- `translation.file_path`
- `handoff.seo.primary_keyword`
- `handoff.seo.secondary_keywords`

## Output

The skill may produce:

- direct edits to the translation file
- a review note with SEO findings
- title, description, slug, heading, and internal-link recommendations

The skill should keep translation meaning intact. Translation quality judgment
belongs to `quality-skills`.

## Tool

The deterministic baseline tool lives under `tools/`.

```bash
python skills/seo/tools/simple_seo_report.py \
  --manifest reports/pr-130/manifest.yaml \
  --target-root target-repo \
  --output reports/pr-130/seo-report.md
```
